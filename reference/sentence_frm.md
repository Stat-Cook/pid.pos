# A short data frame of free text including PID. Used for basic examples and tests.

A short data frame of free text including PID. Used for basic examples
and tests.

## Usage

``` r
sentence_frm
```

## Format

A data frame with 5 rows and 4 columns:

- ID:

  The row number (integer)

- Sentence:

  The free text to detect PID in.

- Numeric:

  Example numeric data (discrete) to be ignored by the algorithm

- Random:

  Example numeric data (continuous) to be ignored by the algorithm
