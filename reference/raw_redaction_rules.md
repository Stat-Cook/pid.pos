# raw_redaction_rules An example of a redaction rules produced by the \`data_frame_report\` function. It is made using the first 20 rows of \`the_one_in_massapequa\` data set.

- If:

  The text to be redacted (character)

- From:

  The text to be replaced (character)

- To:

  The text to replace it with (character)

## Usage

``` r
raw_redaction_rules
```

## Format

An object of class `pid_report` (inherits from `tbl_df`, `tbl`,
`data.frame`) with 10 rows and 3 columns.
