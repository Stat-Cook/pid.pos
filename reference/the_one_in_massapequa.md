# The One in Massapequa

A subset of the 'Friends' data set containing the scene, utterance,
speaker and text of the episode 'The One in Massapequa' (s8e18).

## Usage

``` r
the_one_in_massapequa
```

## Format

A data frame with 3,100 rows and 6 columns:

- scene:

  The scene number (integer)

- utterance:

  The utterance-by-scene number (integer)

- speaker:

  The speaker of the utterance (character)

- text:

  The text of the utterance (character)
